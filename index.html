Problems related to network design naturally arise in the analysis of network connectivity that need to design so that thus could provide not only the transmission of specified volumes during normal operation of the network but also accomplish this with the possible failure of some of its elements. In this regard, the actual scientific task is to develop effective software for data analysis that will solve the problem of designing robust networks that can help enterprises reduce costs through optimal networking.

Taking into account the failure of the arbitrary edge network guarantees its operation in practice with high reliability. This problem may occur, particularly in the creation of local reliably operating internet networks for any enterprise with the least cost.
